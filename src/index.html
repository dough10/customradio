    radio.txt creator

    A website for creating a custom radio text file for use with Hiby digital audio players.

    To use the site:
    Filter stations by genre (ie. hiphop, jazz) then add all the stations you would like included in your radio.txt.
    When you are happy with your list press download and save the text file to the root directory of the storage for your Hiby player.
